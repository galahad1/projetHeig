# Description du contenu du dossier #

* **"EnTeteDeFichier.txt"** contient l'en-tête à insérer et compléter dans les fichiers
contenant du code de l'application développée.

* **"EtapesDeProjetApplication.pdf"** contient une description des grandes étapes d'un projet de
développement d’une application.

* **"ModeleCahierDesCharges.dotx"** contient une structure de base pour le cahier des charges
d'un projet de développement d’une application.

* **"ModeleDossierDeProjetApplication.dotx"** contient une structure de base complète pour la description
et le suivit d'un projet de développement d’une application.

* **"ModeleFicheSignaletique.dotx"** contient une structure de base pour une fiche décrivant le
projet et l'application dans les grandes lignes.

* **"Modèle Journal De Bord.dotx"** contient contenant une structure de base pour un journal de bord
