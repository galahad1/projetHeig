SmartCity v1.0
===============  
Auteurs: Camilo Pineda Serna, Jérémie Zanone, Loan Lassalle, Luana Martelli, Tano Iannetta et Wojciech Myszkorowski  
Dans le cadre du Module Projet 2017 à l'HEIG-VD


Contenu du dossier  
---------------  
1. `README.md` : ce fichier.  
2. `SmartCity.jar` : l'archive de l'application.  
3. `smartcity_schema_data.sql` : le script de création et de peuplement de la base de données.  
3. `smartcity_data.sql` : le script pour uniquement la partie peuplement de la base de données.  
4. `smartcity_schema.sql` : le script pour uniquement la partie création de la base de données.  

Instalation  
---------------  
1. Configurer la base de données :  
	* Exécutez le script MySQL `smartcity_schema_data.sql` pour créer et peupler la base de données.
2. Lancer l'application :  
	* double cliquez sur l'archive `SmartCity.jar` OU  
	* entrez la commande `java -cp ./SmartCity.jar ch.smartcity.SmartCity` à l'emplacement de l'archive, depuis un terminal.
